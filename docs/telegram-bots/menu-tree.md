# Telegram Menu Tree

## Customer Bot

Language first:

- `🇰🇭 ខ្មែរ`
- `🇺🇸 English`

If website or order context exists:

- `✅ Confirm Order`
- `💬 Contact Sales`
- `❌ Cancel`

If no context exists:

- `🛍 Start Order`
- `🔗 Paste Short Order ID`
- `💬 Contact Support`

## Sales Bot

If order context exists:

- `✅ Confirm Details`
- `📦 Ask About Stock`
- `🚚 Change Delivery / Pickup`
- `📝 Add Note`
- `👤 Contact Human Sales`
- `❌ Cancel`

If no order context exists:

- `🔗 Paste Order ID`
- `📋 Paste Order Text`
- `💬 Ask About Product`
- `👤 Contact Human Sales`

## Admin Bot

Entry menu:

- `🆕 New Orders`
- `⏳ Pending`
- `🔄 Processing`
- `✅ Completed`
- `🔎 Search Order`
- `📊 Reports`

Per-order actions:

- `✅ Accept`
- `❌ Reject`
- `❓ Clarify`
- `🔄 Processing`
- `☑️ Complete`
- `🚫 Cancel`
- `📝 Custom Reason`
- `🗒 Add Note`
- `🔁 Refresh`
